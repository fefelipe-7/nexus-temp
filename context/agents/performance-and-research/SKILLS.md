## Fluxo de Ativação

1. Tarefa de performance ou pesquisa.
2. Identificar domínio principal.
3. Carregar skills específicos.

**Se for performance:**
4. Verificar se baseline existe; estabelecer se não.
5. Identificar métrica-alvo (cold start, fps, RAM, bateria, tamanho).
6. Medir antes, aplicar otimização, medir depois.
7. Documentar com data, dispositivo e condições.

**Se for pesquisa:**
4. Definir objetivo e hipótese.
5. Escolher metodologia (quantitativa, qualitativa, A/B).
6. Garantir consentimento e anonimização.
7. Conduzir com diversidade de perfis.
8. Relatório com recomendações acionáveis.

**Se for ambos:**
4. Executar performance primeiro.
5. Medir impacto via pesquisa depois.
6. Correlacionar métricas de performance com satisfação.
