# Skills do Product Behavior

## Skills de Análise de Negócio

| Skill | Quando Usar |
|-------|-------------|
| **startup-business-analyst — Market Sizing** | Estimar TAM/SAM/SOM do Nexus, validar tamanho do problema |
| **startup-business-analyst — Competitive Landscape** | Analisar concorrentes (Notion, Obsidian, Day One, YNAB) |
| **startup-business-analyst — Team Composition** | Avaliar se o ecossistema de agentes cobre as competências |
| **startup-business-analyst — Startup Metrics Framework** | Definir métricas de sucesso do produto |
| **startup-business-analyst — Financial Modeling** | Modelar cenários de sustentabilidade financeira (futuro) |

## Skills de Analytics

| Skill | Quando Usar |
|-------|-------------|
| **business-analytics — KPI Dashboard Design** | Projetar dashboards de relações entre módulos |
| **business-analytics — Data Storytelling** | Apresentar descobertas de produto baseadas em dados |

## Skills de Design e Identidade

| Skill | Quando Usar |
|-------|-------------|
| **brandkit** | Criar/refinar identidade visual do Nexus (logo, paleta, tipografia) |
| **gpt-taste** | Decisões de design direction, layouts bento grids, tipografia editorial |
| **stitch-design-taste** | Curadoria entre variações de design, definir direção visual final |
| **design-taste-frontend** | Aplicar o design system em implementação Flutter |
| **high-end-visual-design** | Polish final de telas importantes |
