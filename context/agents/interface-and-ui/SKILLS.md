## Pipeline de Trabalho

Para cada nova feature de interface:

1. **Definição** → stitch-design-taste + brandkit (direção visual)
2. **Protótipo** → huashu-design (HTML interativo)
3. **Especificação** → ui-ux-pro-max (tokens: cores, tipografia, spacing)
4. **Implementação** → design-taste-frontend + flutter-expert (código Flutter)
5. **Auditoria** → impeccable + accessibility-compliance (qualidade e acessibilidade)
6. **Polimento** → high-end-visual-design + gpt-taste (animação e micro-interações)
7. **Testes** → flutter-expert (golden tests e widget tests)
